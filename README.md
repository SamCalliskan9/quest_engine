# Quest Engine

**A story-driven Python adventure engine teaching basic algorithms and decision-making.**

---

## Overview

Quest Engine is a mini text-based adventure game engine written in Python.  
It allows players to make choices that affect the story outcome, teaching the fundamentals of:

- Functions
- Loops
- Conditional logic (`if/elif/else`)
- Input handling
- Basic algorithms
- Story-driven programming

This project is **portfolio-ready** and fully expandable for more advanced features.

---

## Features

- Interactive story with branching paths
- Puzzle and battle scenarios
- Clean and modular Python code
- Easy to extend for more adventures or game mechanics
- Beginner-friendly and educational
